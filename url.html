What is URL?

A URL (Uniform Resource Locator) is the complete web address used to locate a specific resource on the Internet. Every web page, image, video, or file on the Internet has a unique URL. It tells your browser exactly where to find a resource and how to access it. A URL is what you type into a browser's address bar.

Parts of a URL

A URL has several parts: the Protocol (https://), which defines how data is transferred; the Domain Name (www.example.com), which identifies the website; the Path (/page/about), which points to a specific page; and Query Strings (?id=5), which pass additional data. Understanding each part helps you navigate the web more effectively.

Absolute vs Relative URLs

An absolute URL contains the full address including the protocol and domain (e.g., https://www.example.com/page). A relative URL only contains the path, relative to the current page (e.g., /about or ../images/photo.jpg). Relative URLs are commonly used in HTML code within a website.

URL Encoding

URLs can only contain certain characters. Special characters (like spaces, symbols, or non-ASCII letters) must be encoded using percent-encoding. For example, a space becomes %20 and & becomes %26. This ensures URLs are transmitted correctly across the Internet without misinterpretation.
