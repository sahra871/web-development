What is World Wide Web (WWW)?

The World Wide Web (WWW), commonly known as 'the Web,' is a collection of linked web pages and resources and resources accessed via the Internet using a web browser. It was invented by Sir Tim Berners-Lee in 1989 while working at CERN. The Web is NOT the same as the Internet — the Internet is the infrastructure, while the Web is one of many services that runs on top of it.

How the Web Works

When you type a web address into your browser, it sends a request to a web server. That server responds by sending back HTML, CSS, and other files that your browser renders into the visual web page you see. This happens using the HTTP or HTTPS protocol.

Web vs Internet

Many people confuse the Web with the Internet, but they are different things. The Internet is the global network of connected computers. The World Wide Web is a service (a collection of websites and pages) that runs on the Internet. Other Internet services include email, file transfer (FTP), and online gaming.

Web Technologies

The Web is built on three core technologies: HTML (HyperText Markup Language) for structuring content, CSS (Cascading Style Sheets) for styling and layout, and JavaScript for interactivity. Together, these technologies create the rich, dynamic websites we use every day.
