Welcome to Internet Terminology

This site explains the core concepts behind how the Internet works — from the basic infrastructure to the protocols that power every web page you visit.

🌍

Internet

The global network connecting billions of devices worldwide.
🕸️

World Wide Web (WWW)

The system of linked documents accessed via the Internet.
🖥️

Web Browser

Software used to access and navigate web pages.
🔗

URL

Uniform Resource Locator — the address of a web page.
📡

HTTP / HTTPS

Protocols that define how data is transferred on the web.
🏢

ISP

Internet Service Provider — companies that connect you to the Internet.
🔢

IP Address

A unique numerical label assigned to every device on a network.
⚡

Bandwidth

The maximum rate of data transfer across a network connection.
📖

DNS

Domain Name System — the Internet's address book.
🛡️

Firewall

A security system that monitors and controls network traffic.
